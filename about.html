About SkyCast

A Smart Real-Time Weather Dashboard with a Behavioral Recommendation Engine built with HTML, CSS & JavaScript using the Open-Meteo API (100% free, no API key required).

Real-Time Data

Powered by Open-Meteo — live temperature, humidity, wind speed, UV index, air quality and more.

Smart Engine

Conditional logic converts raw data into behavioral suggestions, clothing tips, and health alerts.

Location Aware

Auto-detects GPS location or searches any of 200,000+ cities worldwide.

Dynamic Themes

Colors change based on weather and time — sunny, rainy, stormy, snowy, night modes.

Responsive

Fully responsive across mobile, tablet and desktop using modern CSS Grid and Flexbox.

Customisable

Settings page lets you control theme, units, accent colour, font size, default city, and more.
